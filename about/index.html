📖 Tentang Jetpack.id

Jetpack.id adalah inisiatif terbuka untuk membangun ekosistem plugin, tema, dan produk digital WordPress yang dibuat oleh developer Indonesia, untuk dunia.

🎯 Misi

Menjadi pusat plugin ringan, dokumentasi profesional, dan kolaborasi terbuka yang bisa dibanggakan oleh komunitas WordPress Indonesia.

📌 Dimulai dari JetForm Connect

Plugin pertama kami, JetForm Connect, menjadi pondasi untuk membangun ekosistem plugin open source yang terstruktur, rapi, dan mudah dikembangkan bersama.

🛤️ Roadmap

  • JetForm Connect → Plugin formulir ringan
  • Jetpack.id → Website pusat dokumentasi & demo
  • Jetpack.id/plugins → Galeri plugin resmi
  • Jetpack.id/themes → Tema WordPress ringan
  • Jetpack.id/collab → Sistem kolaborasi terbuka
  • Jetpack.id/pro → Versi premium (jika dibutuhkan)
