🛤️ Roadmap Jetpack.id

Inilah perjalanan dan rencana besar Jetpack.id — dimulai dari plugin ringan, menuju ekosistem terbuka dan berkelanjutan.

  1. Q1 2025 – JetForm Connect Dirilis

    Plugin formulir ringan pertama Jetpack.id resmi tersedia di WordPress.org.

  2. Q2 2025 – Website Jetpack.id Live

    Halaman plugin, demo, dokumentasi, dan sistem kolaborasi diluncurkan.

  3. Q3 2025 – Tema WordPress Jetpack.id

    Dirilis 2–3 tema ringan pertama dengan desain bersih dan responsif.

  4. Q4 2025 – JetForm Connect PRO

    Versi premium untuk pengguna lanjutan, dengan fitur seperti logika form, integrasi lanjutan, dan statistik.

  5. 2026 – Ekspansi Plugin & API Jetpack.id

    Plugin baru, API publik, dan sistem dashboard kolaborasi mulai dibangun secara bertahap.
